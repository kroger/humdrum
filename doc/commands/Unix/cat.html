COMMAND

 cat -- concatenate files

SYNOPSIS

cat [-entv] [inputfile1 inputfile2 ...]

DESCRIPTION

 The cat command concatenates files into a single output stream. Each file in the list of input files is joined end to end in the order specified on the command line. The output is sent to the standard output or to an output file if file-redirection (`>') is indicated. For example, the command
cat file1 file2
cause a copy of file1 to be displayed on the screen followed by file2

If `-' is given as one of the filenames, then cat will read the standard input.

OPTIONS

 The cat command provides the following options:
-eoutput `$' at the end of each line
-nnumbers all output lines
-toutput tabs as ^I
-vverbose: display all unprintable characters
--helpdisplays a help screen summarizing the command syntax
Options are specified in the command line.

EXAMPLE

 Suppose that a file named file1 contains the following two lines:
1
2
and a second file, file2 contains the following three lines:
A
B
C
The command
cat file1 file2
produces:
1
2
A
B
C
The command
cat -n file2 file1
produces:
1 A
2 B
3 C
4 1
5 2

PORTABILITY

 All UNIX systems supporting the DOS 2.0 and up, with UWIN or MKS Toolkit. Apple OSX.
